导入图像

图像导入、图像添加与图像移除操作指导.

支持格式

格式描述
BMPWindows Bitmap
JPGJoint Photographic Experts Group
PNGPortable Network Graphics
PGMPortable Graymap
TIFFTagged Image File Format
TIFTagged Image File Format

导入图像

通过菜单栏[导入\导出]\[导入图像...]或工作区[导入图像]按钮进行图像导入.

  • 图像导入时,之前图像将被从工作环境中移除.

导入文件夹

通过菜单栏[导入\导出]\[导入文件夹...]进行文件夹导入.

  • 导入打开文件夹中识别格式图像,不包含子文件夹中内容;
  • 导入文件夹中图像时,之前图像将被从工作环境中移除.

从文本列表导入

通过菜单栏[导入\导出]\[从列表导入...]进行图像导入.

  • 根据文本列表解析规则解析打开文本进行图像导入;
  • 导入图像时,之前图像将被从工作环境中移除.

添加图像

通过菜单栏[导入\导出]\[添加图像...]或工作区[添加图像]按钮进行图像添加.

  • 图像添加时,工作环境中已有图像不会被移除;
  • 添加图像时,若工作环境中已有该图像,则跳过该图像.

添加文件夹

通过菜单栏[导入\导出]\[添加文件夹...]进行文件夹添加.

  • 添加打开文件夹中识别格式图像,不包含子文件夹中内容;
  • 添加文件夹中图像时,工作环境中已有图像不会被移除;
  • 添加图像时,若工作环境中已有该图像,则跳过该图像.

从文本列表添加

通过菜单栏[导入\导出]\[从列表添加...]进行图像添加.

  • 根据文本列表解析规则解析打开文本进行图像添加;
  • 添加图像时,工作环境中已有图像不会被移除;
  • 添加图像时,若工作环境中已有该图像,则跳过该图像.

移除图像

图像移除通过解析文本列表实现,可通过菜单栏[导入\导出]\[从列表导入...]或[导入\导出]\[从列表添加...]间接执行.

通过文本列表编写移除列表方法参考文本列表解析规则.

文本列表解析规则

文本规则

# # 行注释符
# = 行注释符
# - 配置符
# ------------------- 配置符说明 -------------------
# -Add      【模式】添加模式(Default)
# -Remove   【模式】移除模式
# -Path     【配置】文件路径设置(Default: ./)
# -Folder   【项目】添加/移除文件夹（绝对路径）
# -folder   【项目】添加/移除文件夹（相对路径）
# -File     【项目】添加/移除文件（绝对路径名）
# -file     【项目】添加/移除文件（相对路径名）
# ------------------- THE END -------------------
